* Roque es contador, honesto y joven
* Ana es ingeniera y honesta
* Cecilia es abogada

**Nota**: resolverlo de forma que la base se pueda consultar de la siguiente forma:

```prolog
? honesto(Quien).
Quien = roque;
Quien = ana.
```
